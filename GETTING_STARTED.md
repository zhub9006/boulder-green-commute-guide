# 🚀 Getting Started with Green Commuting in Boulder

This guide helps you adapt the commute analysis for your own home-to-office route in Boulder.

## How to Use This Guide

1. **Find your coordinates** using [OpenStreetMap](https://www.openstreetmap.org) or a geocoding service
2. **Run a commute analysis** using OSM routing tools (foot and bike modes)
3. **Analyze your neighborhood** at both ends of your route
4. **Compare modes** using the data templates in this repository

## Tools Used

- **OSM Geocoding** — Convert addresses to coordinates
- **OSM Routing** — Get turn-by-turn directions for walking and biking
- **OSM Neighborhood Analysis** — Discover nearby amenities and livability scores
- **OSM Commute Analysis** — Compare multiple transportation modes

## Data Sources

All data is derived from [OpenStreetMap](https://www.openstreetmap.org), a free, open geographic database maintained by volunteers worldwide.

## Contributing

Want to add more Boulder commuters' analyses? Here's how:

1. Fork this repository
2. Create a new branch: `add-your-name-commute`
3. Add your commute analysis following the template
4. Open a Pull Request

## Template

```markdown
## My Commute: [Home Address] → [Office Address]

- **Distance:** __ km (__ miles)
- **Walking Time:** ~__ minutes
- **Biking Time:** ~__ minutes
- **Best Mode:** [Walking / Biking / Both]
- **Key Streets:** [List main streets along route]
```
