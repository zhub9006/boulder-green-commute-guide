# Contributing Guide — Add Your Own Commute

Boulder is a city of cyclists and walkers! Want to share your own commute route analysis?

1. Fork this repo: `git clone https://github.com/zhub9006/boulder-green-commute-guide.git`
2. Create a branch: `git checkout -b my-commute-route`
3. Add a file in `routes/`: `routes/your_name_commute.md`
4. Use the template: Title, distance/time comparison, segment directions, safety notes, CO₂ savings
5. Commit & push. Open a PR with your route summary.

Data Sources: OpenStreetMap (CC BY-SA), RTD (buses), City of Boulder, B-Cycle.
Be respectful. Fact-check. Cite sources.
