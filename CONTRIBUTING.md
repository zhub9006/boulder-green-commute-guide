# 🤝 Contributing to Boulder Green Commute Guide

We welcome contributions from Boulder residents and anyone interested in sustainable transportation! Here's how you can help:

---

## Ways to Contribute

### 🌍 Add New Routes
Submit commute analyses for other Boulder routes:
1. Origin and destination addresses
2. Walking time and distance
3. Biking time and distance
4. Turn-by-turn directions
5. Nearby amenities
6. Seasonal considerations

### 📊 Add Data
- More routes and neighborhoods
- Transit (RTD) connections and schedules
- Cost comparisons for different modes
- Safety data from city reports

### 🛠️ Fix Issues
- Outdated information
- Broken links
- Typos or unclear instructions
- Missing sections

### 💡 Suggestions
- New content ideas
- Feature requests
- Design improvements

---

## How to Submit

1. **Fork** the repository
2. **Create a branch** for your contribution
3. **Make your changes**
4. **Test** any scripts or data updates
5. **Open a Pull Request** with a clear description

---

## Submission Guidelines

- Use clear, helpful descriptions
- Cite your sources
- Include data tables where helpful
- Be respectful and constructive in PR reviews
- Follow the existing formatting style
