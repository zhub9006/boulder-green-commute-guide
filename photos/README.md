# Photo Library

## Submission Guidelines

- Submit **two photos minimum** per route: start location & end location
- Photos should be **landscape orientation** (16:9)
- Include **seasonal capture date** in filename
- Show **route landmarks** for orientation
- Submit original + compressed versions
