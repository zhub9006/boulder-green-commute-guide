# 📋 Contributing to the Boulder Green Commute Guide

## How to Help Build a Better Guide

This is a community-powered project. Your contributions help make sustainable commuting easier and more appealing for all Boulder residents.

---

## 📝 Adding a New Commute Route

1. **Choose your route** — Home address → Work address (or any two points in Boulder)
2. **Collect data** — Use [OpenStreetMap's routing tool](https://map.project-osrm.org/) or the OSM Valhalla engine to get turn-by-turn directions
3. **Record coordinates** — Note your start and end GPS coordinates
4. **Create a new file** — Name it `commute_<your_street>_to_<Destination>.md`
5. **Include**:
   - Start and end addresses + verified GPS coordinates
   - Distance, time estimates for walking and biking
   - Turn-by-turn directions (each segment)
   - Safety ratings (1–5 stars)
   - Seasonal tips
   - Environmental impact (CO₂ saved, calories burned)
   - Any landmarks or notes
6. **Open a Pull Request** — Describe your new route and why it's useful

---

## 🔧 Editing Existing Data

- **Routing changes?** If Boulder adds bike lanes, removes sidewalks, or changes traffic patterns — update the routing data
- **Seasonal notes need refresh?** Check seasonal tips for currency (e.g., construction on 19th St)
- **Station/list data outdated?** RTD, B-Cycle, and micro-mobility services update frequently

---

## 📸 Adding Photos

- Bike-friendly streetscapes, trail views from Canyon Boulevard
- Screenshots of the OSM routing output for verification
- Photos of bike racks, bus stops, or other infrastructure

---

## 🐛 Reporting Issues

- Broken links
- Outdated coordinates
- Safety hazards reported by users
- New transit routes or schedules

---

## 🤷 Guidelines

- **Be accurate** — Always verify route data against the latest OSM / Valhalla output
- **Be kind** — This guide is for promoting sustainable transport, not shaming car users
- **Be local** — Include Boulder-specific details (street names, neighborhoods, elevation)
- **Cite your sources** — Specify the date and engine used for verification

---

## 📬 Get in Touch

- **Open an issue** — Suggest improvements, new routes, or feature requests
- **GitHub Issues** — Found a bug or outdated data? File an issue
- **Pull Requests** — We welcome contributions from all Boulder residents!
