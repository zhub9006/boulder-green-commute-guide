# Contributing to Boulder Green Commute Guide

Thank you for helping make Boulder's streets greener! Here's how to contribute.

## Adding a New Commute Route

1. **Get coordinates** — Use OSM or leave them as lat/lng
2. **Travel the route** — Walk/bike it at least once during rush hour
3. **Record details** — Use our [route template](routes/template.md)
4. **Add it** — Create a PR with:
   - Route name (e.g., "Sparkill ↔ Downtown")
   - At least 2 photos (start & end)
   - Seasonal variation notes
   - Safety observations

## Format Guidelines

### File Naming
- `routes/{slug}-{mode}.md` — e.g., `routes/sparkill-downtown-walking.md`
- Photos: `photos/{slug}/{season}.jpg`

### Markdown Style
```markdown
## Route Name

- **Distance:** X.X km
- **Time:** X min
- **Difficulty:** Easy/Moderate/Hard
- **Best For:** Morning/Evening/All day
- **Safety Rating:** ⭐⭐⭐/5
```

## Code of Conduct

- Be respectful in route descriptions
- Note accessibility limitations (curbs, stairs, hills)
- Prioritize safety over speed in recommendations
- Acknowledge Indigenous Ute land where routes pass

## PR Review

All submissions reviewed within 7 days by community moderators. Look for:
- ✅ Accurate coordinates and distances
- ✅ Current (2025) infrastructure status
- ✅ Inclusive language and accessibility notes
- ✅ No copyrighted images
